# Retro Snake Game

A classic Snake game with retro arcade style and responsive design.

## Features

- Retro arcade-style design with pixelated graphics
- CRT screen effect with scanlines and glitches
- Responsive layout that adapts to different screen sizes (380px, 430px, 480px, 530px, 630px, 840px and above)
- Mobile-friendly controls with touch buttons
- Swipe gesture support on mobile
- Keyboard controls on desktop
- Score tracking with high score saved in local storage
- Visual and sound effects for enhanced experience

## How to Play

- **On Desktop**: Use arrow keys to change direction
- **On Mobile**: Use on-screen buttons or swipe in the direction you want to move
- Eat food to grow longer and increase your score
- Avoid running into walls or your own tail
- The game gets faster as your score increases

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Bootstrap 5

## Screen Size Support

The game is optimized for various screen sizes including:
- Extra small mobile (380px)
- Small mobile (430px)
- Medium mobile (480px)
- Large mobile (530px)
- Small tablets (630px)
- Tablets and small desktops (840px)
- Larger screens (841px+)

Additionally, specific adjustments are made for both portrait and landscape orientations.

## Implementation Details

- Pixelated food and snake design for authentic retro feel
- Alternating colored snake body segments
- Random "glitch" effects for retro arcade experience
- Dotted grid lines reminiscent of early computer games
- CRT scanline effect that moves down the screen
- Custom responsive design for all target screen sizes
- Special layout for landscape orientation on mobile devices
